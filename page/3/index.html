废弃API取消出现Warning的方法

2016-01-07

对于有代码洁癖的人来说, 出现一个黄色的小小的警告都是无法忍受的, 但是项目中又会由于要向下适配用到旧的API, 不可避免的就会出现警告, 查了资料一直没找到, 所以只好自己动手了

TARGETS -> Build Settings -> Apple LLVM7.0-Warnings-All languages -> Deprecated Functions -> YES 改为NO
然后cmd+B, 这样就OK了, 烦人的警告就不会出现了
